# Smart Invoice Generator (WordPress Plugin)

Smart Invoice Generator is a multi-industry WordPress plugin that allows businesses to create professional invoices tailored to their specific business type.

Unlike generic invoice tools, this plugin dynamically changes invoice fields based on the selected business category (e.g., Doctors, Consultants, Freelancers, Agencies), providing a customized invoicing experience.

## ✨ Features

- Dynamic business-type-based invoice forms
- Custom fields for different industries
- Clean admin interface
- Custom database storage for scalability
- Flexible architecture for future extensions
- Invoice numbering system
- Status management (Paid / Unpaid)

## 🎯 Supported Business Types

- Doctors & Medical Services
- Consultants
- Freelancers
- Agencies
- Service Businesses

## 🚀 Use Cases

- Small and medium businesses
- Professional service providers
- Agencies and freelancers
- Industry-specific billing needs

## 🛠️ Tech Stack

- WordPress Plugin API
- PHP (OOP)
- JavaScript
- MySQL Custom Tables

## 📌 Installation

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin from the WordPress dashboard
3. Start creating invoices from the admin panel
